# Personalized Nutrition Generator

## Overview
An AI-powered nutrition recommendation system that analyzes blood reports using OCR and Machine Learning techniques to generate personalized diet plans and health insights.

## Features
- Blood Report OCR Processing
- Health Parameter Analysis
- Machine Learning Based Predictions
- Calorie Estimation
- Personalized Diet Recommendations
- PDF Report Generation

## Technologies Used
- Python
- Flask
- Machine Learning
- OCR
- MongoDB
- HTML
- CSS
- JavaScript

## Project Workflow
1. Upload Blood Report
2. Extract Data using OCR
3. Analyze Health Parameters
4. Predict Health Insights
5. Generate Personalized Diet Plan
6. Download Report
